# PowerTech — сайт

Одностраничный трёхъязычный сайт PowerTech LLC (диагностика и мониторинг качества
электроэнергии). Статический: чистый HTML + один CSS, без сборки и зависимостей.

## Структура

```
/                 → index.html        — армянская версия (главная, lang="hy")
/en/index.html    → английская версия (lang="en")
/ru/index.html    → русская версия   (lang="ru")
style.css         → общий стиль для всех трёх страниц
assets/           → изображения (.webp), шрифты (Mardoto, .woff2), фон Hero
assets/brand/     → логотипы (SVG/PNG) и подписи для почты
favicon.ico, favicon.svg, apple-touch-icon.png
```

Версия CSS подключается с параметром `?v=NN` (например `style.css?v=141`). **При любом
изменении `style.css` нужно поднять номер во всех трёх HTML** — иначе у пользователей
останется старый стиль из кэша. Изменения только в тексте HTML номер поднимать не требуют.

## Локальный запуск

Любой статический сервер из корня проекта, например:

```bash
python -m http.server 3000
# или
npx serve .
```

Затем открыть `http://localhost:3000/`.

## Деплой

Пути в проекте **абсолютные от корня** (`/style.css`, `/assets/…`, `/en/`, `/ru/`),
поэтому сайт должен раздаваться **с корня домена**:

- ✅ Custom-домен (powertech.am), **Netlify**, **Vercel**, **Cloudflare Pages**,
  или GitHub Pages на user-сайте `username.github.io` — работает как есть.
- ⚠️ GitHub Pages в **подпапке проекта** (`username.github.io/repo/`) — абсолютные
  пути сломаются. Используйте custom-домен или один из хостингов выше.

Сборка не нужна: залить файлы как есть.

## Форма заявки (почта) — подключено

Форма отправляется через **Web3Forms** (без бэкенда). Реализация — AJAX (`fetch`)
без перезагрузки страницы, с инлайн-сообщением об успехе/ошибке и honeypot-защитой
от спама. Заявки приходят на почту, привязанную к Access Key.

- **Access Key** хранится в каждой из трёх форм (`index.html`, `en/index.html`,
  `ru/index.html`) в `<input type="hidden" name="access_key" …>`. Это публичный
  ключ Web3Forms — держать его в HTML штатно и безопасно.
- Сменить адрес получателя или ключ → зарегистрировать новый ключ на
  https://web3forms.com и заменить значение `access_key` во всех трёх формах.
- Тема письма задаётся в `<input name="subject">` (своя на каждый язык).

Альтернативы при желании: **Formspree** (`action="https://formspree.io/f/ВАШ_ID"`),
**Netlify Forms** (при хостинге на Netlify — атрибут `netlify` в `<form>`).
